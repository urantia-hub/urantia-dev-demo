# Contributing

## Prerequisites

- [Node.js](https://nodejs.org/) (v18+)

## Setup

```bash
npm install
cp .env.example .env.local
# Fill in your .env.local values
npm run dev
```

## Submitting Changes

1. Fork the repo
2. Create a feature branch (`git checkout -b my-feature`)
3. Make your changes
4. Open a pull request against `main`
